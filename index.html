Flask Forum

The website is built using both frontend and backend technologies. It allows users to create their own forums, where other users can create posts from existing forums, add comments to posts, and reply to comments that facilitates in open and dynamic conversations.

The application is a Single Page Application (SPA) developed using React and Vue as it's frontend, with a Flask-based REST API powering it's backend. This architecture ensures faster load times by dynamically updating specific components of the page instead of reloading the entire page.

Chess Game

The chess game is developed in C# using the .NET Framework as a Window Forms Application. It supports both Player vs. Player (PvP) and Player vs. AI (PvAI) game modes, with various level of difficulties for the AI opponent.

The AI is powered by the Minimax algorithm, a recursive decision-making strategy. It evaluates all possible future moves and their outcomes, allowing AI to choose the move that offers the most strategic advantage. This is done by recursively traversing the game tree and backtracking where each node compares the values of its child nodes to select the most favorable outcome.

The evaluation is based on a material scoring system, where each chess piece is assigned a specific point value.
